\section{Perfil Profesional}

software y atención a clientes de alto valor. Autor de librerías open-source de alto rendimiento en
Node.js (sin dependencias externas) para análisis de datos y parseo de formatos binarios.
Especializado en desarrollo full-stack con JavaScript/TypeScript, Java y Python; experiencia
construyendo plataformas web complejas, aplicaciones móviles y herramientas de sistemas. Capacidad demostrada para
diseñar soluciones que compiten en rendimiento con ecosistemas consolidados como Polars/Pandas.

%% cervid/decomposer
\entry{\href{https://github.com/CervidLabs/decomposer}{CervidLabs/decomposer} — Intérprete de formatos binarios}{Node.js · Open Source}{}
\begin{itemize}
  \item Intérprete de bajo nivel para formatos binarios científicos y de ML: \texttt{.npy}, \texttt{.npz} (NumPy) y
        \texttt{.parquet} (Apache).
  \item Parseo nativo sin dependencias: implementa de cero los protocolos de serialización de cada formato.
  \item Permite consumir artefactos de Python/PyTorch/scikit-learn directamente desde Node.js, eliminando
        la necesidad de conversión o microservicios puente.
  \item Arquitectura modular: cada formato es un plugin independiente con interfaz unificada.
\end{itemize}
\vspace{0.2em}
\textcolor{muted}{\small \texttt{Node.js · TypeScript · Binary Parsing · NumPy · Apache Parquet}}

\begin{itemize}
  \item Decoder JPEG propio (SOF0/SOF1/SOF2 progressive) con Huffman paralelo por waves e IDCT en
